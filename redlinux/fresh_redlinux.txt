How To Make A Fresh Redlinux Install: (32 or 64 bits)

1. Install the latest version of ArchLinux whether 32 or 64 bits.
2. Login as root and run the following commands:
wget http://redlinernotes.com/docs/redlinux/pkglists/pkglist.txt
pacman -S `cat pkglist.txt`
3. Add new user (guest), add to sudoers. Login as guest.
4. Extract and insert default config files:
wget http://redlinernotes.com/docs/redlinux/configs.tar.gz
tar -zxvf configs.tar.gz
mv configs/* ~/
sudo mv ~/etc/* /etc/*
5. Run scripts/addrepos.sh && pacman -S paktahn. Add .paktahn/config.lisp with "(in-package :pak) (setf *color-scheme* *color-scheme-lightbg*)".
6. Run the following commands:
wget http://redlinernotes.com/docs/redlinux/assets.tar.gz
tar -zxvf assets.tar.gz
mv wallpapers images/wallpapers
mv Crashbit .icons/Crashbit
wget http://redlinernotes.com/docs/redlinux/pkglists/aurlist.txt
paktahn -S `cat aurlist.txt`
7. Add "vboxdrv" to modules in rc.conf, add sshd hal wicd alsa @slim @mpd @openntpd to daemons. !network too, I think.
8. Add halt, reboot, pm-suspend and gparted to /etc/sudoers.
9. Hand-install any necessary larch stuff...and LARCH IT. Note: Larch only supported on 32 bit systems at this time.
10. Test it, document stuff, test it, document stuff, larch it, update the changelog, release it! Files with guest in them: .asdf-install, .sbclrc, .eclrc, mpd.conf, etc.
